# Swiggy Sales Analysis Dashboard

## Overview
This project analyzes Swiggy sales data using Excel to identify trends, customer behavior, and business performance.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Slicers

## KPIs
- Total Orders
- Total Revenue
- Average Order Value
- Top Performing City

## Key Insights
- Peak order time occurs during evening hours
- Certain cities contribute highest revenue
